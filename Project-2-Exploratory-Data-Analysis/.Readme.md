# Project 2 – Exploratory Data Analysis (EDA) using Python

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on an e-commerce sales dataset using **Python**. The objective is to understand the dataset, analyze customer purchasing patterns, identify trends, detect outliers, and extract meaningful business insights through statistical analysis and data visualization.

The project was completed as part of a Data Analytics Internship and demonstrates the practical application of Python for data analysis.

---

## 🎯 Objectives

* Perform exploratory data analysis on a cleaned dataset.
* Calculate descriptive statistics for numerical features.
* Analyze data distributions and identify outliers.
* Examine product performance and revenue generation.
* Study customer payment preferences and order status.
* Identify monthly sales trends.
* Explore relationships between numerical variables using correlation analysis.
* Derive business insights to support data-driven decision-making.

---

## 🛠️ Tools & Libraries Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Dataset Information

The dataset contains e-commerce transaction records with the following information:

* Order Details
* Customer Information
* Product Information
* Quantity Purchased
* Unit Price
* Total Price
* Payment Method
* Order Status
* Referral Source
* Shipping Details

The dataset consists of **1,200 records** and was verified to contain **no missing values** or **duplicate records**, making it suitable for analysis.

---

## 📈 Analysis Performed

The following analyses were carried out during the project:

* Data Overview and Structure
* Data Quality Assessment
* Descriptive Statistics
* Distribution Analysis

  * Quantity
  * Unit Price
  * Total Price
* Product Analysis

  * Most Ordered Products
  * Revenue by Product
* Payment Method Analysis
* Order Status Analysis
* Monthly Sales Trend
* Correlation Analysis using Heatmap
* Business Insights and Conclusion

---

## 🔍 Key Insights

* Customer purchases are generally concentrated around lower order quantities, while bulk purchases occur less frequently.
* Product demand varies across the catalog, with some products contributing significantly more revenue than others.
* Online payment methods are widely preferred by customers.
* Order statuses highlight opportunities to further improve order fulfillment and customer experience.
* Monthly sales exhibit noticeable fluctuations, indicating seasonal purchasing behavior.
* Correlation analysis shows that **Total Price** is strongly influenced by **Unit Price** and **Quantity**.
