# SQL Data Analysis using MySQL

## Project Overview

This project demonstrates SQL-based data analysis on an e-commerce sales dataset using MySQL Workbench. The objective was to extract meaningful business insights by writing SQL queries using filtering, sorting, grouping, and aggregate functions.

The dataset used in this project was cleaned during Project 1 and explored during Project 2 before being imported into MySQL for analysis.

---

## Tools Used

- MySQL Workbench
- SQL
- Microsoft Excel (Data Preparation)

---

## Dataset

The dataset contains e-commerce sales records with information such as:

- Order ID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Payment Method
- Order Status
- Referral Source
- Total Price

## Business Questions Answered

Some of the analyses performed include:

- Total number of orders
- Total revenue generated
- Average order value
- Highest and lowest order values
- Revenue by payment method
- Quantity sold by product
- Orders by referral source
- Top spending customers
- Best-selling products
- Monthly sales revenue

---

## Key Insights

- Identified the highest revenue-generating payment methods.
- Determined the top-performing products based on sales quantity.
- Calculated the average order value.
- Analyzed customer spending patterns.
- Generated monthly sales summaries.
- Used grouping and aggregate functions to answer business-related questions.
